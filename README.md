# ribosomespeedcontrol

This repository contains Raw Data and analysis scripts for our current bioRxiv preprint "Around one fifth of yeast mRNAs form a separately addressable, translation elongation-controlled regulon" (<a href='https://www.biorxiv.org/content/10.1101/513705v1' target=_blank>Kazana et al. 2019</a>). Analysis scripts are provided in Jupyter Notebook format.
